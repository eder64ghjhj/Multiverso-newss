# 🌌 Multiverso News

Portal de notícias temático desenvolvido do zero com HTML5 e CSS3, com páginas dedicadas aos universos de **Fórmula 1**, **Dragon Ball** e **Cavaleiros do Zodíaco**. Projeto criado de forma independente durante o programa PROA, sem seguir template ou tutorial — cada página foi construída com identidade visual própria.

## 📰 Páginas do Portal

- **Fórmula 1** — Notícias e cobertura do mundo da F1
- **Dragon Ball** — Conteúdo do universo Dragon Ball
- **Cavaleiros do Zodíaco** — Cobertura do clássico anime

## ✨ Funcionalidades

- Navegação entre páginas temáticas
- Layout com identidade visual por tema
- Uso de imagens e assets personalizados
- Estrutura semântica HTML5

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
